About This Repo

If you need to write or read TinCanAPI statements and state in an iOS or OSX application, this is the repo for you. We intend to make this SDK spec complete over time and welcome any additions from the community. While we plan to keep this repo current, there is no support for it from Rustici Software. Feel free to ask questions and post issues, but the priority will typically be low.

This SDK code should work fine on iOS 5+ and OS X as well, but it is new code and will rely on the community for most of the testing. Please open an issue if you find something a little off.
